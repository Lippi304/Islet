<objective>
Make the collapsed idle island merge invisibly with the physical notch. Today the black
collapsed pill is sized from the hardcoded constant `NotchPillView.collapsedSize =
CGSize(width: 200, height: 38)`, while the real notch on this machine measures ~179x32pt
(and differs on other notch MacBooks). So even in a RELEASE build (pure black, offset 0)
the idle form overhangs / frames the hardware notch instead of disappearing into it.

The app ALREADY measures the true notch: `notchSize(...)` returns the real CGSize and
`NotchWindowController.positionAndShow(on:)` computes the real `collapsedFrame`. The fix
threads that measured size into `NotchPillView.collapsedIsland`'s `.frame(...)`, with a
sane fallback to the current 200x38 when no notch is measured.

Purpose: deliver D-01 (idle-invisible / "unsichtbar mit der Hardware-Notch verschmelzen").
Output: a data-driven collapsed pill size, DEBUG dev aids preserved, verified on-device.
</objective>
