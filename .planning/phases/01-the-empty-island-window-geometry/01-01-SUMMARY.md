## Final Pure-Function Signatures (Plan 02 consumes these verbatim)

```swift
// NotchGeometry.swift
func hasNotch(safeAreaTop: CGFloat, auxLeftWidth: CGFloat?, auxRightWidth: CGFloat?) -> Bool
func notchSize(screenWidth: CGFloat, safeAreaTop: CGFloat,
               auxLeftWidth: CGFloat?, auxRightWidth: CGFloat?,
               widthFudge: CGFloat = 4) -> CGSize?
func notchFrame(screenFrame: CGRect, safeAreaTop: CGFloat,
                auxLeftWidth: CGFloat?, auxRightWidth: CGFloat?,
                widthFudge: CGFloat = 4) -> CGRect?

// DisplayResolver.swift
struct ScreenDescriptor: Equatable {
    let uuid: String?
    let frame: CGRect
    let safeAreaTop: CGFloat
    let auxLeftWidth: CGFloat?
    let auxRightWidth: CGFloat?
    let isBuiltin: Bool
}
func selectTargetScreen(from screens: [ScreenDescriptor]) -> ScreenDescriptor?
```

`widthFudge` default = **4**. No compiler disambiguation alias was needed: the module-qualified `Islet.hasNotch(safeAreaTop:auxLeftWidth:auxRightWidth:)` call inside `ScreenDescriptor.hasNotch` resolves to the free function cleanly (the illustrative `Islet_hasNotch` alias from the plan was not used).

## Decisions Made
- `widthFudge` default 4 (boring.notch convention) for a seamless hardware overlap.
- Built `ScreenDescriptor.hasNotch` as a computed property delegating to the free `hasNotch` via `Islet.hasNotch(...)` — keeps one source of truth for the notch rule and avoids name shadowing without renaming the geometry function.
- nil from `selectTargetScreen` is the contract for "hide the panel" (clamshell/external-only/non-notch), not an error path.
