```swift
import SwiftUI

enum ActivitySettings {
    // @AppStorage / UserDefaults keys — used by BOTH SettingsView and the controller (Plan 04).
    static let chargingKey   = "activity.charging"
    static let nowPlayingKey = "activity.nowPlaying"
    static let deviceKey     = "activity.device"
    static let accentIndexKey = "accentIndex"

    // D-12 curated palette (~5-6 swatches). Index 0 = neutral default (D-12).
    static let palette: [Color] = [.white, .blue, .green, .orange, .pink, .purple]
    static let defaultAccentIndex = 0

    // Map a persisted index → the concrete accent Color (clamped — out-of-range falls to neutral).
    static func accent(for index: Int) -> Color {
        palette.indices.contains(index) ? palette[index] : palette[defaultAccentIndex]
    }
}

// 06-RESEARCH §Pattern 4 — the single accent source the three lively leaf views read.
private struct ActivityAccentKey: EnvironmentKey { static let defaultValue: Color = .white }
extension EnvironmentValues {
    var activityAccent: Color {
        get { self[ActivityAccentKey.self] }
        set { self[ActivityAccentKey.self] = newValue }
    }
}
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user → UserDefaults (@AppStorage) | The user sets four small local preferences (three Bools + one Int index) stored in `UserDefaults.standard`. These are local app preferences, NOT secrets and NOT a privilege boundary — any process running as the user can already read/write them. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-07 | Tampering | A tampered/out-of-range `accentIndex` in UserDefaults | mitigate | `ActivitySettings.accent(for:)` clamps any out-of-range index back to the neutral default (`palette.indices.contains` guard), so a corrupted value can never index out of bounds or crash. |
| T-06-08 | Information disclosure | Secrets stored in @AppStorage/UserDefaults | accept | No secrets are stored here — only three booleans and an integer index. UserDefaults is not used for any credential (notary creds live in the keychain, handled by release.sh, not this plan). Documented: nothing sensitive is persisted. |

No threat reaches **high** severity: the only persisted data is non-sensitive local preferences, and the one parse path (accent index) is bounds-clamped. The security gate passes.
</threat_model>
