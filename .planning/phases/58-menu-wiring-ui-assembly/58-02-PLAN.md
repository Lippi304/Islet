| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-58-05 | Information Disclosure | `confirmDeleteAllHistory` | mitigate | Must call BOTH `clipboardStore.clear()` and `ClipboardFileStore.save([], ...)` â€” confirmed by acceptance criteria grep on Task 1; leaving either call out means the encrypted-at-rest guarantee is undermined by data the user was told is gone (RESEARCH.md Pitfall 2, PITFALLS.md Security Mistakes table) |

| T-58-07 | Information Disclosure | Pasteboard-access explanation's persisted `UserDefaults` flag | accept | The flag only records "has this alert been shown," never any clipboard content itself â€” no sensitive data stored in `UserDefaults` by this plan |
