<objective>
Build the shelf's pure data model (`ShelfItem`, `ShelfLogic`), its session-temp file-copy
mechanic (`ShelfFileStore`), and the thin `ShelfCoordinator` that wires the two together —
isolated, Foundation-only, unit-tested Swift — no AppKit, no `NSItemProvider`, no drag APIs.

Purpose: Establish SHELF-08's session-only guarantee (never persisted to disk) and the
append/remove/clear/dedupe lifecycle contract before any fragile drag/panel/view code
(Phases 20-22) is touched. `ShelfCoordinator` closes D-05 fully within this phase: it is the
one place that actually calls `ShelfFileStore.deleteSessionCopy` the moment an item leaves the
shelf (individual removal or clear-all), so SHELF-08's "never persisted" guarantee is a real,
tested enforcement mechanism today — not a hook a later phase might forget to wire. Mirrors
this project's own established convention (`IslandResolver`/`TransientQueue` proven pure before
`NotchWindowController` wiring; `DeviceCoordinator` — a thin class that owns side effects and
calls into a pure reducer — proven in isolation before Phase 16's controller wiring).

Output: `Islet/Shelf/ShelfItem.swift`, `Islet/Shelf/ShelfLogic.swift`,
`Islet/Shelf/ShelfFileStore.swift`, `Islet/Shelf/ShelfCoordinator.swift`, plus their test
files — all consumed for the first time by Phase 20's view/controller wiring (out of scope
here; Phase 20 only needs to call `append`/`remove`/`clear` on the coordinator instance it
owns, including from an eventual app-quit hook — the deletion mechanism itself is already
correct and tested).

shape `ShelfLogic` must mirror):
```swift
struct TransientQueue {
    private(set) var head: ActiveTransient?
    private var pending: [ActiveTransient] = []
    let maxDepth = 2

    mutating func enqueue(_ t: ActiveTransient) -> Bool {
        if head == nil { head = t; return true }
        if head == t || pending.contains(t) { return false }
        pending.append(t)
        if pending.count > maxDepth { pending.removeFirst() }
        return false
    }

    mutating func removeAll(where predicate: (ActiveTransient) -> Bool) {
        pending.removeAll(where: predicate)
        if let h = head, predicate(h) {
            head = pending.isEmpty ? nil : pending.removeFirst()
        }
    }
}
```
