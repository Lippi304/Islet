<objective>
Build the IPC + state layer of the `NowPlaying*` quartet: a `@Published NowPlayingState` model
(the SwiftUI-facing holder, mirroring `ChargingActivityState`) and a thin `NowPlayingMonitor`
that wraps the adapter's `MediaController` (mirroring `PowerSourceMonitor`). This is the SINGLE
isolated MediaRemote seam (CLAUDE.md mandate: swapping the implementation later is a one-file change).

The monitor consumes ONE persistent stream (`onTrackInfoReceived` + `startListening()`), lifts each
payload into a plain `TrackSnapshot` for the Plan-01 pure seam, hands the pre-decoded `NSImage?`
artwork to the model, sends transport down the same child's stdin, and synthesizes the D-12
launch-time health check the wrapper does not expose. It tears the child down on deinit (no
orphaned perl).

Purpose: Quarantine the fragile private-MediaRemote bridge behind one small, well-defined file so
a future Apple break is a one-file fix, and so the health-state machine (D-12 launch-fail vs D-13
mid-drop) is implemented in exactly one place with the correct, researched semantics.
Output: A startable/stoppable monitor + an observable state model that Plan 03 (view) and Plan 04
(controller) consume. No views and no controller wiring here.
</objective>

    Implement:
    ```swift
    import MediaRemoteAdapter
    import AppKit

    @MainActor
    final class NowPlayingMonitor {
        private let controller = MediaController()
        // nil snapshot = "no media now" (engine emitted NIL). Non-nil = a track update. NSImage is the pre-decoded artwork.
        private let onSnapshot: (TrackSnapshot?, NSImage?) -> Void
        private let onTerminated: () -> Void   // D-13 mid-session death

        init(onSnapshot: @escaping (TrackSnapshot?, NSImage?) -> Void,
             onTerminated: @escaping () -> Void) {
            self.onSnapshot = onSnapshot
            self.onTerminated = onTerminated
        }

        func start() {
            controller.onTrackInfoReceived = { [weak self] info in
                guard let self else { return }
                guard let p = info?.payload else { self.onSnapshot(nil, nil); return }   // NIL → no media (D-11)
                let snap = TrackSnapshot(bundleIdentifier: p.bundleIdentifier,
                                         isPlaying: p.isPlaying, title: p.title,
                                         artist: p.artist, hasArtwork: p.artwork != nil)
                self.onSnapshot(snap, p.artwork)   // artwork already off-thread-decoded by the wrapper
            }
            controller.onListenerTerminated = { [weak self] in self?.onTerminated() }   // D-13
            controller.startListening()            // ONE persistent `loop` child — survives restart: emits current session immediately
        }

        // Mirror PowerSourceMonitor.stop(): terminate the child + clear the handler. Called from the controller's deinit (Plan 04) — no orphaned perl (T-04-04).
        func stop() { controller.stopListening() }

        // NOW-02 — transport rides the EXISTING child's stdin (no re-spawn):
        func togglePlayPause() { controller.togglePlayPause() }
        func nextTrack()       { controller.nextTrack() }
        func previousTrack()   { controller.previousTrack() }

        // D-12 launch-time health check (Pattern 3 option a). The wrapper does NOT expose
        // ungive's `test` subcommand, so synthesize: getTrackInfo's nil is ambiguous
        // (no-media OR failure — Pitfall 1), so treat "a callback arrived at all" as healthy
        // and "no callback within the timeout" as unavailable. onListenerTerminated does NOT
        // fire for a never-emitted child (eventCount==0 — Pitfall 2), so this probe is what
        // covers the blocked-at-launch case.
        func runHealthCheck(then setHealthy: @escaping (Bool) -> Void) {
            var settled = false
            controller.getTrackInfo { info in
                if settled { return }; settled = true
                setHealthy(true)
            }
            DispatchQueue.main.asyncAfter(deadline: .now() + 3.0) {
                if settled { return }; settled = true
                setHealthy(false)   // never heard back → D-12 "nicht verfügbar"
            }
        }
    }
    ```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| perl child process → app | The adapter spawns ONE `/usr/bin/perl` child streaming newline-delimited JSON over stdout. The app reads/decodes that (untrusted) stream. The wrapper does the framing + decode; the monitor only consumes the already-decoded `TrackInfo?`. |
| untrusted media metadata → model | title/artist/bundleIdentifier come from arbitrary playing apps. They are copied as optional Strings into the model — no command construction, no format string, no file write. |
| untrusted artwork bytes → NSImage | Album art is decoded inside the wrapper (`NSImage(data:)`); invalid bytes yield `artwork == nil` → the monitor reports `hasArtwork: false` → Plan 03's placeholder path. No raw byte handling in our code. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-04 | Denial of Service (resource leak) | Orphaned/zombie `perl` child after teardown | mitigate | `stop()` calls `controller.stopListening()` (terminates the child + clears the readability handler). Plan 04's controller calls it from `deinit`, mirroring `powerMonitor.stop()` discipline (T-03-06). |
| T-04-05 | Tampering / DoS | Malformed / oversized JSON line from the child | mitigate | The wrapper decodes per-line with `JSONDecoder` and routes failures to `onDecodingError` without crashing; the monitor reads every payload field as already-optional and never force-unwraps. Bounds/truncation of title/artist on DISPLAY are Plan 03's job. |
| T-04-06 | Tampering | Untrusted artwork bytes → image decode | mitigate | Decode happens inside the wrapper; invalid image → `artwork == nil` → `hasArtwork: false` → placeholder. The monitor never touches raw bytes. |
| T-04-07 | Availability (silent failure of a private bridge) | MediaRemote blocked at launch (entitlement gone on this macOS) | mitigate | `runHealthCheck` times out at 3s → `isHealthy=false` → D-12 "nicht verfügbar"; `onListenerTerminated` handles mid-session death (D-13). A broken bridge degrades gracefully, never crashes (success-criterion-3). |
| T-04-08 | Elevation of Privilege | The perl child running with a private-framework entitlement | accept | The bypass is Apple's OWN platform-binary (`/usr/bin/perl`) entitlement, invoked with fixed args from the embedded `run.pl` — no user-controlled command construction, no privilege our code grants. Out of our control surface. |
</threat_model>
