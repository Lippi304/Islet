# Phase 18 ‚Äî UI Design Contract

> Visual and interaction contract for the song-change toast. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

**Scope note:** This is a native macOS SwiftUI overlay, not a web app ‚Äî the shadcn/Tailwind
gate and the 60/30/10 web-surface color model do not apply. This contract adapts the template
to the project's existing design system (a single black "island" blob with SF-Symbols glyphs
and system-font text), verified by reading `NotchPillView.swift`/`NotchWindowController.swift`
directly rather than asking the user to re-specify values the codebase already locks in.

## Spacing Scale

Declared values (existing app-wide scale, multiples of 4 ‚Äî confirmed by reading
`NotchPillView.swift`'s actual `.padding(...)` calls):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-text gaps |
| sm | 8px | Compact element spacing (e.g. `EqualizerBars`/`ProgressBar` internal spacing) |
| md | 16px | Default horizontal content padding (`expandedIsland` uses `.padding(.horizontal, 16)`) |
| lg | 24px | Section padding |
| xl | 32px | Top notch/camera-clearance band (`mediaExpanded`'s `.padding(.top, 32)`) |
| 2xl | 48px | Major section breaks (not used by this phase) |
| 3xl | 64px | Page-level spacing (not used by this phase) |

**Toast-specific values:**
- Horizontal content inset: **16px** (md) ‚Äî matches `expandedIsland`'s pattern, the closest
  existing "downward blob with no side art" precedent.
- Title‚Üîartist vertical gap: **2px** ‚Äî a documented sub-4 exception, mirroring the existing
  `mediaExpanded` title/artist `VStack(spacing: 1)` precedent (line ~504 of
  `NotchPillView.swift`); the toast uses `spacing: 2` (not 1) since its text is
  center-aligned/no leading art crowds it, still a deliberate tight-kerning exception, not the
  base 4pt scale.

Exceptions: 2px title/artist gap (tight text-stack kerning, matches existing in-app precedent for the same title/artist pairing).

## Typography

**ROUND 3 (final, on-device feedback):** the toast is a single compact text row under the
wings, not a title/artist pair at `mediaExpanded`'s own sizes ‚Äî one combined line reads
smaller:

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Toast row (title ‚Äî artist, one `Text`) | 12px | medium (500) | 1.2 (SwiftUI system default; single line, no wrap) |
| Label | not used by this phase | ‚Äî | ‚Äî |
| Display | not used by this phase | ‚Äî | ‚Äî |

Uses `design: .rounded` (matches every other text element in `NotchPillView.swift`).
`.lineLimit(1)` + `.truncationMode(.tail)` ‚Äî a toast is a brief transient glance, not a
paragraph; a long combined "title ‚Äî artist" string truncates rather than wraps or grows the
shape taller.

## Color

The app has no 60/30/10 web-page surface split ‚Äî it is a single black island blob with text/
icon accents. Adapted mapping:

| Role | Value | Usage |
|------|-------|-------|
| Dominant | `Color.black` | The island blob itself (`blobShape`'s `.fill(Color.black)`) ‚Äî always, including the toast |
| Secondary | `Color.white` / `.secondary` (SwiftUI semantic, adapts light/dark text-on-black) | Round 3 (final): the combined toast text row uses `.foregroundStyle(.white)` throughout (a single `Text`, no separate secondary-colored artist span). |
| Accent (persisted user swatch, `\.activityAccent`) | user-selected, e.g. green/blue/etc. | **NOT used by the toast.** Reserved exclusively for the three existing "lively leaf elements": charging bolt glyph, `EqualizerBars`, device glyph/battery fill, `ProgressBar` fill. The toast's title/artist text stays white/secondary ‚Äî same "untinted chrome" rule already applied to `mediaExpanded`'s own title/artist (D-10/D-05 precedent: "never accent-tinted"). |
| Destructive | not applicable | No destructive actions in this phase |

Accent reserved for: charging bolt glyph (`wings(for:)`), `EqualizerBars` fill, device glyph/battery fill (`deviceWings`), `ProgressBar` fill (`mediaExpanded`). The song-change toast is explicitly excluded from this list.

| Auto-dismiss duration | ~3.0s, matching the existing `activityDuration` seed (do not hardcode a new duplicate `3.0` literal elsewhere) | `NotchWindowController.swift` line 165, CONTEXT.md/ROADMAP "~3s" |
