<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| IOKit power-source read → app | The app reads a CFDictionary of power-source values supplied by the OS. Keys/values may be missing or malformed mid-transition. No network, no user input, no persistence. |
| `@convention(c)` callback → Swift object | The C run-loop callback recovers the `PowerSourceMonitor` instance through a raw `context` pointer — a lifetime boundary (use-after-free risk if the source outlives the object). |
| Notification thread → main thread / @Published / AppKit | The callback may run off-main; mutating `@Published`/AppKit there is a data race. |
