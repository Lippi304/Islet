# Requirements: Notch — Dynamic Island for Mac

**Defined:** 2026-07-02
**Core Value:** The notch becomes a beautiful, reliable "island" that shows now-playing media and reacts when you plug in the charger or connect a device — it must feel native, smooth, and as polished as the iPhone Dynamic Island.

## v1.0.1 Requirements

Pre-release polish for the still-unreleased v1.0 build. Each maps to roadmap phases.

### Now Playing Progress Bar (PBAR)

- [x] **PBAR-01**: User sees a horizontal progress bar in the expanded Now Playing view showing playback position, with elapsed/remaining time labels (e.g. "1:23 / 3:45"); updates smoothly while playing, holds still while paused; no tap-to-seek
  - **Outcome:** Shipped in Phase 7 exactly as scoped. On-device UAT surfaced and fixed a pause-transition backward-flash bug (drift-corrected freeze estimate); a code review closed a NaN/Infinity crash risk. Verified passed (7/7 truths, `07-VERIFICATION.md`).

### Fullscreen Stability (FS)

- [x] **FS-01**: Entering true fullscreen shows no visible island flash at any point during or after the transition
  - **Outcome:** Resolved via Phase 9 after Phase 8's investigation of an alternative candidate signal (CGS event 106/107) was disproven on-device and formally escalated (`08-ESCALATION.md`). Phase 9's Candidate C — a dedicated max-level private CGS Space, additive to the existing `.canJoinAllSpaces` behavior — eliminated the flash on the first wave, confirmed on-device across all 3 trigger methods with zero regressions. Verified passed (8/8 must-haves, `09-VERIFICATION.md`). One non-blocking follow-up (CR-01: CGS Space leak on app quit) tracked as tech debt, not a requirement gap.

## v2 Requirements

Deferred to a future release (v1.0 backlog, unchanged — carried forward, not re-scoped this milestone).

### Later Features

- **SHELF-01**: File shelf — drag-and-drop tray at the notch to temporarily hold files, then drag them back out / share / AirDrop
- **HUD-01**: System HUDs — replace the default volume / brightness / battery overlays with notch-based HUDs
- **TMR-01**: Timer — start and watch a countdown timer as a live activity in the island
- **DIST-01**: Real Developer-ID notarization (pending Apple Developer account purchase)
