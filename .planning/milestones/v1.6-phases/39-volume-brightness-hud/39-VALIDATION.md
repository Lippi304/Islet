`xcodebuild test` is known to hang on this project — the test host boots the full `Islet.app` (NSPanel/MediaRemote/IOBluetooth all initialize), per this project's recorded memory (`xcodebuild-test-headless-hang`). Do not use `xcodebuild test` as an automated gate for this phase's plans; route test execution to manual Cmd-U.
